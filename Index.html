Basic HTML and HTML5

The Language for building web pages. HTML is a markup language that uses a special syntax or notation to describe the structure of a webpage to the browser. HTML element usually have opening and closing tags that surround every content

Basic CSS

CSS or Cascading Style Sheets, tell the browser how to display the text and other content that you write in HTML. With CSS, you can control the color, font, size, spacing and manny other aspects of HTML elements.
